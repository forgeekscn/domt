# 学生宿舍管理系统 (DOMT)

## 项目简介

这是一个基于J2EE开发的学生宿舍管理系统，主要用于高校学生宿舍的日常管理工作，包括学生信息管理、班级管理、楼栋管理、宿舍管理等功能模块。

## 功能特性

- 学生信息管理：添加、编辑、删除、查询学生基本信息
- 班级管理：班级信息的增删改查，班级学生管理
- 楼栋管理：宿舍楼栋信息维护
- 宿舍管理：宿舍信息维护，学生住宿分配
- 用户权限管理：系统管理员、宿舍管理员等不同角色权限控制
- 数据统计分析：学生住宿情况统计，空房统计等

## 技术栈

- 后端框架：Spring MVC + MyBatis
- 前端技术：JSP + JavaScript + CSS
- 数据库：MySQL
- 服务器：Tomcat
- 开发工具：Eclipse/IDEA

## 项目结构

```
src/
  └── cn/forgeeks/
      ├── controller/    # 控制器层
      ├── service/       # 业务逻辑层
      ├── dao/           # 数据访问层
      ├── domain/        # 实体类
      ├── mapper/        # MyBatis映射文件
      ├── pagination/    # 分页相关
      └── util/          # 工具类
WebRoot/                 # Web资源目录
```

## 部署说明

1. 克隆项目到本地
2. 导入数据库脚本（位于项目文档中）
3. 配置数据库连接参数（jdbc.properties）
4. 部署到Tomcat服务器
5. 访问系统登录页面

## 开发环境

- JDK 1.8+
- MySQL 5.7+
- Tomcat 8.5+
- Maven 3.6+
